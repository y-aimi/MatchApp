# MatchApp
タイマー機能付き神経衰弱アプリ

2ペア×8組=16枚が配置されています。

# DEMO

![result.gif](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/592467/8aba9d6a-5d51-3e2b-8e09-c82810d03f8e.gif)

# Features
・タイマー機能

・タイマー終了時にアラート機能

・画面遷移によるリスタート機能

・フリップアニメーションの付与

・AVFoundation.frameworkで効果音付与
